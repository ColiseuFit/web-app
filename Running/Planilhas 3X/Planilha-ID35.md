# Planilha ID 35

## Nível Intermediário: Acelerando nos 3km (Meta 3km)

### 1. Configurar Nova Planilha (Painel Admin)
*Espelho da primeira tela de cadastro no sistema.*

- **Título da Planilha:** Nível Intermediário: Acelerando nos 3km
- **Nível:** Intermediário
- **Duração (Semanas):** 4
- **Freq. Semanal:** 3
- **Descrição Breve (Objetivo):** Uma periodização ágil e de alta performance de 4 semanas desenvolvida para corredores intermediários que buscam quebrar barreiras de tempo nos 3km. Utilizando 3 sessões semanais perfeitamente divididas (Intervalado para velocidade, Tempo Run para resistência de ritmo e Longão para base de fôlego), o aluno atingirá o Pace Alvo de 5:15 min/km no teste final.
